# WarrantyBox - 保修管家

**版本**: 1.0.0  
**最低系统要求**: iOS 16.0+  
**定价**: $0.99

---

## 📱 应用简介

WarrantyBox 是一款极简的保修凭证管理工具，帮助您：
- 📷 拍照存储购买凭证（发票、保修卡）
- 📝 记录商品信息、购买日期、保修期限
- 🔔 保修到期前主动提醒（30 天/7 天/1 天）
- 📂 按类别分类管理
- 🔍 快速搜索商品或商家

**隐私安全**: 所有数据本地存储，无需注册，不上传云端。

---

## 🎯 核心功能

| 功能 | 说明 |
|------|------|
| 凭证录入 | 拍照或从相册导入购买凭证 |
| 信息管理 | 商品名称、类别、购买日期、保修期限、商家信息 |
| 到期提醒 | 保修期结束前推送通知 |
| 分类管理 | 电子产品、家用电器、家具、服装、其他 |
| 快速搜索 | 按商品名或商家搜索 |

---

## 🏗️ 技术架构

- **UI 框架**: SwiftUI
- **数据存储**: CoreData
- **通知系统**: UserNotifications
- **最低版本**: iOS 16.0

---

## 📁 项目结构

```
WarrantyBox/
├── WarrantyBoxApp.swift      # 应用入口
├── ContentView.swift         # 主界面
├── AddItemView.swift         # 添加项目界面
├── Models.swift              # 数据模型
├── DataController.swift      # CoreData 控制器
├── NotificationManager.swift # 通知管理器
├── Assets.xcassets/          # 资源文件
└── CoreData/                 # CoreData 模型
```

## 🚀 快速开始

1. 打开 `WarrantyBox.xcodeproj`
2. 选择目标设备（iPhone/iPad 模拟器或真机）
3. 点击运行 (⌘R)
