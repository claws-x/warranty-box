# WarrantyBox Review Response Templates

## 1. Why do you request notification permission?

WarrantyBox uses notification permission only to send local reminders before a user's warranty expires. Permission is requested only when the user chooses to enable reminder functionality.

## 2. Why do you request camera permission?

Camera permission is used only when the user chooses to photograph a receipt or product image and attach it to a warranty record.

## 3. Why do you request photo-library permission?

Photo-library permission is used only when the user chooses an existing receipt or product image to attach to a warranty record.

## 4. Does the app collect or upload personal data?

In the current version, WarrantyBox stores warranty records locally on the user's device and does not automatically upload this data to a remote server.

## 5. Does the app require an account?

No. WarrantyBox does not require account registration or third-party login in the current version.
