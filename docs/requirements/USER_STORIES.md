## US-001: 添加保修凭证

### 故事描述
作为 **用户**  
我想 **拍照添加保修卡**  
以便 **保存购买凭证，防止丢失**

### 详细需求

#### 功能流程
```
1. 用户点击首页「+」按钮
2. 选择「拍照」或「从相册选择」
3. 拍摄/选择保修卡图片
4. 填写产品信息表单
5. 点击保存
6. 返回首页，新凭证显示在列表顶部
```

#### 表单字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 产品名称 | 文本 | ✅ | 如「iPhone 15 Pro」 |
| 品牌 | 文本/选择 | ✅ | 如「Apple」 |
| 购买日期 | 日期选择器 | ✅ | 默认今天 |
| 保修期限 | 数字 + 单位 | ✅ | 如「1 年」、「24 个月」 |
| 到期日期 | 日期选择器 | ✅ | 自动计算，可手动调整 |
| 分类 | 单选 | ✅ | 电器/数码/家具/服装/其他 |
| 备注 | 文本 | ❌ | 可选填写 |
| 图片 | 图片 | ✅ | 至少 1 张 |

#### 验收标准
- [ ] 可以从相机拍照
- [ ] 可以从相册选择图片
- [ ] 可以填写所有表单字段
- [ ] 到期日期自动计算 (购买日期 + 保修期限)
- [ ] 保存后在列表中可见
- [ ] 保存后可以继续添加下一个

## US-002: 查看保修列表

作为 **用户**  
我想 **查看所有保修凭证列表**

- [ ] 列表显示产品缩略图、名称、品牌

- [ ] 可以按类别筛选

## US-003: 到期提醒

### 故事描述
作为 **用户**  
我想 **在保修到期前收到提醒**  
以便 **及时处理维修/退换事宜**

### 详细需求

#### 提醒设置
```
设置 > 通知 > 提醒时间

☑️ 30 天前提醒
☑️ 7 天前提醒
☐ 1 天前提醒
```

#### 通知内容
```
📱 推送标题：WarrantyBox 提醒
📝 推送内容：您的「iPhone 15 Pro」保修将在 30 天后到期 (2026-04-30)
```

#### 验收标准
- [ ] 支持设置多个提醒时间点 (30 天/7 天/1 天)
- [ ] 本地推送通知
- [ ] 点击通知跳转到对应凭证详情
- [ ] 可以在设置中开启/关闭提醒
- [ ] 每个凭证独立提醒
- [ ] 已过期凭证不再提醒

## US-006: 数据导出

### 故事描述
作为 **用户**  
我想 **导出保修凭证**  
以便 **备份或分享给家人**

### 详细需求

#### 导出格式
| 格式 | 说明 | 适用场景 |
|------|------|----------|
| PDF | 包含所有信息和图片 | 打印、存档 |
| 图片 | 原始保修卡图片 | 快速分享 |

#### 导出流程
```
1. 选择凭证 (单个或批量)
2. 点击「导出」按钮
3. 选择导出格式 (PDF/图片)
4. 选择分享方式 (微信/邮件/存储)
```

#### 验收标准
- [ ] 支持导出为 PDF
- [ ] 支持导出为图片
- [ ] 可以选择单个或批量导出
- [ ] 导出文件可以分享到其他应用
- [ ] PDF 包含完整信息 (名称、品牌、日期、图片)
- [ ] 批量导出时生成合并 PDF

## 验收测试用例

前置条件：已添加一个保修凭证，到期日期为 30 天后

1. 进入设置 > 通知

3. 等待或模拟时间到 30 天前
预期结果：
- 收到本地推送通知
- 通知内容正确显示产品名称和到期日期
- 点击通知跳转到凭证详情
