## 📱 产品概述

### 产品名称
**WarrantyBox** - 保修凭证管理工具

### 产品定位
一款帮助用户集中管理所有商品保修凭证的 iOS 应用，通过拍照存档、智能提醒、分类管理等功能，让用户不再丢失保修卡，及时享受售后服务。

### 核心价值
- 📸 **拍照存档** - 纸质保修卡电子化，永不丢失
- ⏰ **到期提醒** - 保修期前智能提醒，不错过维权时机
- 📂 **分类管理** - 按类别/品牌/购买日期快速查找
- 🔒 **隐私安全** - 本地存储，数据完全私有

### 目标平台
- iOS 17.0+
- iPhone (主力)
- iPad (适配)

## 📖 用户故事

### US-001: 添加保修凭证
```
作为 用户
我想 拍照添加保修卡
以便 保存购买凭证，防止丢失

验收标准:
- 可以从相机拍照
- 可以从相册选择图片
- 可以填写产品名称、品牌、购买日期、保修期限
- 可以设置保修到期日期
- 保存后在列表中可见
```

### US-002: 查看保修列表
```
作为 用户
我想 查看所有保修凭证列表
以便 了解我有哪些商品的保修信息

验收标准:
- 列表显示产品缩略图、名称、品牌
- 显示保修到期剩余天数
- 可以按购买日期/到期日期排序
- 可以按类别筛选
```

### US-003: 到期提醒
```
作为 用户
我想 在保修到期前收到提醒
以便 及时处理维修/退换事宜

验收标准:
- 支持设置多个提醒时间点 (30 天/7 天/1 天)
- 本地推送通知
- 点击通知跳转到对应凭证详情
- 可以在设置中开启/关闭提醒
```

### US-004: 搜索保修凭证
```
作为 用户
我想 快速搜索保修凭证
以便 在需要时快速找到

验收标准:
- 支持按产品名称搜索
- 支持按品牌搜索
- 支持按类别搜索
- 搜索结果实时显示
```

### US-005: 分类管理
```
作为 用户
我想 自定义分类管理保修凭证
以便 更有条理地组织信息

验收标准:
- 预设分类：电器、数码、家具、服装、其他
- 可以添加自定义分类
- 可以编辑/删除分类
- 可以将凭证分配到不同分类
```

### US-006: 数据导出
```
作为 用户
我想 导出保修凭证
以便 备份或分享给家人

验收标准:
- 支持导出为 PDF
- 支持导出为图片
- 可以选择单个或批量导出
- 导出文件可以分享到其他应用
```

## 🔍 竞品分析

### 主要竞品

| 竞品 | 优势 | 劣势 | 我们的差异化 |
|------|------|------|--------------|
| **Google Keep** | 免费、多平台 | 非专业保修管理、无提醒 | 专业保修管理、到期提醒 |
| **Evernote** | 强大笔记功能 | 付费、复杂 | 简洁专注、一次性付费 |
| **各类扫描 App** | 扫描功能强 | 无保修管理逻辑 | 专为保修设计、分类+ 提醒 |
| **纸质保管** | 无需学习 | 易丢失、难查找 | 电子化、易搜索、有提醒 |

### 市场机会
1. **专注细分** - 目前没有专门的保修管理主流应用
2. **隐私优先** - 本地存储，不上传云端，符合隐私趋势
3. **简洁易用** - 相比通用笔记应用，更专注更简单
4. **合理定价** - $0.99 一次性付费，门槛低

## 🎨 用户体验要求

### 设计原则
1. **简洁** - 3 步内完成凭证添加
2. **直观** - 图标和文字清晰易懂
3. **高效** - 列表加载<1 秒
4. **可靠** - 数据不丢失，操作有反馈

### 界面规范
- 遵循 iOS Human Interface Guidelines
- 支持浅色/深色模式
- 支持动态字体
- 支持 VoiceOver 无障碍

### 关键流程
```
添加凭证流程 (≤30 秒):
1. 点击「+」按钮
2. 拍照/选图
3. 填写信息 (名称、品牌、购买日期、保修期)
4. 保存

查看到期流程 (≤5 秒):
1. 打开应用
2. 首页列表查看剩余天数
3. 红色高亮显示即将到期
```

## 🔒 隐私与安全

### 数据存储
- **存储位置**: 本地 CoreData
- **图片存储**: 本地沙盒
- **备份**: 支持 iCloud 备份 (用户选择)

### 隐私承诺
- 不收集用户个人信息
- 不上传用户数据到服务器
- 不包含第三方追踪 SDK
- 不包含广告

### 权限说明
| 权限 | 用途 | 必需 |
|------|------|------|
| 相机 | 拍摄保修卡 | 是 (可选相册) |
| 相册 | 选择已有图片 | 是 (可选相机) |
| 通知 | 到期提醒 | 是 (可关闭) |
